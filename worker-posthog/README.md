# worker-posthog (legacy)

Ce dossier contient l'ancien reverse proxy PostHog basé sur Cloudflare Worker.

## Statut

- Deprecated / legacy.
- Ne plus utiliser en configuration standard.
- Le chemin principal utilise désormais le Managed Reverse Proxy PostHog avec `api_host: https://festibask-impro.fr`.

## Usage autorisé

Uniquement pour rollback ou diagnostic ponctuel.
